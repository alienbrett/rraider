Setting up
===========

Installing
----------

Install from pypi:

.. code-block:: bash
   
   pip install rraider


Or manually:

.. code-block:: bash
   
   git clone https://github.com/alienbrett/rraider
   cd rraider
   pip install -e .


Testing
--------


Run the tests quickly and easily

.. code-block:: bash
   
   python3 -m unittest
