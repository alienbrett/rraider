Data¶

rraider.data.get(keyword, cache=True, **kwargs)¶

Acquires some data source.

Parameters

  • keyword – keyword that identifies the dataset to be returned. Could be {‘tsy’}

  • cache – whether to stash this dataset for future use. Cache invalidates each day, so results are alway from today.

Returns

pandas datetime instance.

Examples

type( ttools.data.get('tsy') )
# pandas.core.frame.DataFrame

ttools.data.get('tsy', floatDates=True)
# This one will have column names as the number of years until maturity
